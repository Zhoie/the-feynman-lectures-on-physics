# Recoil Momentum in a Finite Release Window (SI)

Two equal carts are released by a finite-duration impulse pair (not an
instantaneous ideal kick). Validation focuses on the pre-wall window to match
real apparatus measurements.

## What you can do
- Change **mass**, **release impulse**, and **release duration**.
- In **Advanced**, add release asymmetry and fixture friction.

## What to notice
- Pre-wall total momentum drift should stay below 1% in calibrated mode.
- Center-of-mass drift should stay below 0.5% of track length.
- Energy stability is checked in a pre-wall observation window.
